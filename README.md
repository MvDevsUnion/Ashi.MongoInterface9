# Ashi.MongoInterface9
internal library made originally for Ashi based on https://medium.com/@marekzyla95/mongo-repository-pattern-700986454a0e   
new features has been added on top of the original fork for this project  
this abstract layer makes it easier for talking to mongo db and normalize models

supported for .net 9 LTS
can be downgraded manually to .net 8 as well
